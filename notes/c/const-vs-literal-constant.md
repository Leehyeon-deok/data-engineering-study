# C: const Variable vs Literal Constant

## 1. Literal Constant (리터럴 상수)

### 정의
- 소스 코드에 **직접 작성된 값 자체**
- 이름이 없으며 변수처럼 메모리에 저장되지 않음

### 예시
```c
10
3.14
'A'
"Hello"
c
코드 복사
int a = 10;   // 10은 리터럴 상수
특징
이름 없음

값 변경 불가

메모리 주소 없음

자료형은 존재함

값 그 자체를 의미

2. const Variable (const 변수)
정의
읽기 전용 변수

초기화 후 값 변경 불가

메모리에 실제로 존재

예시
c
코드 복사
const int MAX = 10;
특징
이름 있음

메모리에 저장됨

주소 연산자(&) 사용 가능

값 변경 불가

3. 차이점 정리
구분	const 변수	리터럴 상수
이름	있음	없음
메모리 존재	있음	없음
주소(&)	가능	불가능
값 변경	불가능	불가능
정체	변수	값

4. 중요 포인트 (시험 자주 출제)
c
코드 복사
const int a = 10;
printf("%p", &a);   // 가능
c
코드 복사
printf("%p", &10);  // 컴파일 에러
리터럴 상수는 메모리 주소를 가질 수 없음
