Java Basic Concepts Summary
1. 상수 (Constant)

변하지 않는 값

자바에서는 final 예약어를 사용하여 선언한다.

final int MAX = 100;

2. 리터럴 (Literal)

프로그램에서 직접 사용하는 값

숫자, 문자, 문자열, 논리값 등을 모두 포함한다.

int a = 10;        // 정수 리터럴
char c = 'A';      // 문자 리터럴
boolean b = true; // 논리 리터럴

3. 형 변환 (Type Casting)
3.1 묵시적 형 변환 (자동 형 변환)

작은 자료형 → 큰 자료형

데이터 손실 없음

byte bNum = 10;
int iNum = bNum;

3.2 명시적 형 변환 (강제 형 변환)

큰 자료형 → 작은 자료형

데이터 손실 가능

int iNum = 10;
byte bNum = (byte) iNum;

4. 논리 연산자

조건식에서 사용

연산자	의미
&&	두 항이 모두 참일 때 참
`	
!	참 → 거짓, 거짓 → 참
5. 조건 연산자 (삼항 연산자)

if–else 문을 한 줄로 표현 가능

조건식 ? 결과1 : 결과2;

int max = (a > b) ? a : b;

6. 비트 논리 연산자

비트 단위로 연산 수행

연산자	설명
&	두 비트가 모두 1이면 1
`	`
^	서로 다르면 1, 같으면 0
~	0 → 1, 1 → 0 (비트 반전)
7. 비트 이동 연산자
연산자	설명
<<	왼쪽으로 비트 이동
>>	오른쪽으로 비트 이동 (부호 비트 유지)
>>>	오른쪽으로 비트 이동 (부호와 무관하게 0으로 채움)
8. if–else 문과 조건 연산자 비교
if (a > b)
    max = a;
else
    max = b;

max = (a > b) ? a : b;

9. do–while 문

조건과 상관없이 최소 1번은 실행

do {
    실행문;
} while (조건식);

10. continue 문

반복문에서 사용

현재 반복을 건너뛰고 다음 반복으로 이동

for (int i = 1; i <= 5; i++) {
    if (i == 3) continue;
    System.out.println(i);
}


출력:

1
2
4
5
