🌐 네트워크 계층별 장비 및 핵심 개념 정리
1️⃣ 네트워크 기기의 계층별 분류

네트워크 기기는 OSI 7계층을 기준으로 처리 범위를 나눌 수 있다.

계층	주요 장비
애플리케이션 계층 (L7)	L7 스위치
인터넷 계층 (L3)	라우터, L3 스위치
데이터 링크 계층 (L2)	L2 스위치, 브리지
물리 계층 (L1)	NIC, 리피터, AP
2️⃣ 애플리케이션 계층 (L7)
🔹 L7 스위치 (Application Load Balancer)

여러 장비를 연결하여 통신을 중재

목적지가 연결된 포트로만 전기 신호 전송

로드 밸런서라고도 불림

주요 기능

서버 부하 분산

클라이언트 요청을 여러 서버로 분배

시스템이 처리 가능한 트래픽 증가

필터링 기능 제공

장애 서버 자동 제외

헬스 체크

일정 주기로 서버 상태 확인

3-way handshake 실패 시 비정상 판단

전송 주기·재전송 횟수는 서버 부하를 고려해 설정

🔹 L4 스위치 vs L7 스위치
구분	L4 스위치	L7 스위치
기준	IP, Port	IP, Port + URL, HTTP 헤더, 쿠키
스트리밍/메시지 인식	X	O
분산 정밀도	낮음	높음
클라우드 예시

L7 로드 밸런싱 → ALB

L4 로드 밸런싱 → NLB

🔹 서버 이중화

로드 밸런서는 2대 이상의 서버 필수

장애 발생 시 자동 전환

안정적인 서비스 제공 목적

3️⃣ 인터넷 계층 (L3)
🔹 라우터 (Router)

여러 네트워크를 연결·분할·구분

서로 다른 네트워크 간 통신 가능

패킷 소모 최소화

최적 경로 탐색 후 패킷 포워딩

🔹 L3 스위치

L2 스위치 + 라우팅 기능

하드웨어 기반 라우팅

고속 처리 가능

4️⃣ 데이터 링크 계층 (L2)
🔹 L2 스위치

장치들의 MAC 주소 관리

MAC 주소 테이블 기반 스위칭

IP 주소 이해 ❌

라우팅 기능 ❌

🔹 브리지 (Bridge)

두 개의 LAN을 상호 연결

트래픽 분리 및 확장

5️⃣ 물리 계층 (L1)
🔹 NIC (Network Interface Card)

컴퓨터를 네트워크에 연결

물리적 신호 송수신

🔹 리피터 (Repeater)

신호 증폭

전송 거리 확장

🔹 AP (Access Point)

유선 LAN을 무선 LAN으로 변환

무선 신호 제공 전용

라우팅 기능 ❌

6️⃣ 핵심 네트워크 프로토콜 및 개념
🔹 ARP / RARP

ARP: IP → MAC 주소 변환

RARP: MAC → IP 주소 변환

🔹 홉 바이 홉 통신 (Hop-by-Hop)

라우팅 테이블을 기반으로

다음 목적지(IP)로 단계적 이동

패킷이 최종 목적지까지 전달되는 과정

즉, 라우팅 과정

🔹 라우팅 테이블

목적지 네트워크 정보

해당 목적지로 가는 방법 저장

라우터 내부에 존재

🔹 게이트웨이 (Gateway)

서로 다른 통신망·프로토콜 간 연결

네트워크의 출입문 역할

7️⃣ IP 주소 체계
🔹 IP 버전

IPv4

IPv6

🔹 클래스 기반 주소 할당

A / B / C / D / E 클래스

앞부분: 네트워크 주소

뒷부분: 호스트 주소

8️⃣ DHCP (Dynamic Host Configuration Protocol)

IP 주소 및 통신 매개변수 자동 할당

인터넷 접속 시 자동 IP 설정

가정용 공유기·라우터에 기본 포함

9️⃣ NAT (Network Address Translation)

사설 IP ↔ 공인 IP 변환

IP 주소를 다른 주소로 매핑

🔹 장점

공인 IP 절약

내부 네트워크 보호

🔹 단점

다수 사용 시 속도 저하 가능

실제 접속 호스트 수 증가에 따라 성능 저하

✅ 한 줄 요약 (시험용)

L7/L4 스위치 → 부하 분산

라우터 → 네트워크 간 연결

L2 스위치 → MAC 기반 스위칭

AP → 무선 신호 제공

ARP → IP → MAC

NAT → 주소 변환
