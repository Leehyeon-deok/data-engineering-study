🧠 1. 응집도 (Cohesion) = “모듈 내부”

👉 한 모듈 안이 얼마나 잘 뭉쳐 있나

✔ 높을수록 좋음 (좋은 설계)

🔥 응집도 순서 (중요!)

👉 시험용 암기 순서

우논시절통순기
📌 응집도 7단계 (낮 → 높)
단계	이름	핵심 키워드
1	우연적 (Coincidental)	그냥 모아둠
2	논리적 (Logical)	같은 종류 묶음
3	시간적 (Temporal)	시간 기준 (초기화 등)
4	절차적 (Procedural)	순서대로 실행
5	통신적 (Communicational)	같은 데이터 사용
6	순차적 (Sequential)	출력 → 입력 연결
7	기능적 (Functional)	하나의 기능만 수행 ⭐
💣 응집도 암기 핵심
우연 → 논리 → 시간 → 절차 → 통신 → 순차 → 기능
(뒤로 갈수록 좋다)

👉 최고의 응집도 = 기능적

🧠 2. 결합도 (Coupling) = “모듈 간”

👉 모듈끼리 얼마나 서로 의존하나

✔ 낮을수록 좋음 (좋은 설계)

🔥 결합도 순서 (중요!)

👉 시험용 암기

내공외제스자
📌 결합도 6단계 (강 → 약)
단계	이름	핵심
1	내용 결합도 (Content)	내부 직접 변경
2	공통 결합도 (Common)	전역변수 공유
3	외부 결합도 (External)	외부 환경 공유
4	제어 결합도 (Control)	flag로 제어
5	스탬프 결합도 (Stamp)	구조체 일부 사용
6	자료 결합도 (Data)	필요한 데이터만 전달 ⭐
💣 결합도 암기 핵심
내공외제스자
(내부 → 외부 → 약해짐)

👉 최고의 결합도 = 자료 결합도
