🔹 3. Java 다형성 (중요🔥)

🔹 12. for문 동작 구조
for(초기식; 조건식; 증감식){
    실행문;
}
✔ 실행 순서
초기식 → 조건식 → 실행문 → 증감식 → 반복
✔ 주의
for(...);  // ← 세미콜론

👉 실행문 없음 (버그)

🔹 13. 프로세스 vs 스레드
구분	설명
프로세스	CPU에 의해 실행되는 프로그램
스레드	프로세스 내부의 실행 흐름 (가벼움)
✅ 한줄 핵심 암기 요약
생성자: 부모 먼저, super 자동
실행: new 기준 (다형성)
스케줄링: 선점 = 빠름 / 비선점 = 단순
DB: 통저운공
보안: 공급망 / 부채널 / 제로데이
Python: 타입 없음 + 들여쓰기 중요
