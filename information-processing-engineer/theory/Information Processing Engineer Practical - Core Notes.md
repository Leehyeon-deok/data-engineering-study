# 📘 Information Processing Engineer Practical - Core Notes

## 1. OLAP와 데이터웨어하우스(DW)

* **데이터웨어하우스(DW)**
  대량의 데이터를 저장하고 관리하는 시스템

* **OLAP (온라인 분석 처리)**

  * DW 또는 데이터 마트와 연계됨
  * 데이터를 **전략적 정보로 변환**
  * **의사결정 지원 역할**

---

## 2. 시맨틱 웹과 온톨로지

* **시맨틱 웹(Semantic Web)**

  * 인터넷과 같은 분산 환경에서 자원 간 관계 정보를
    **기계가 이해하고 처리할 수 있도록 하는 지능형 웹**

* **온톨로지(Ontology)**

  * 실세계의 개념, 속성, 관계를
    **컴퓨터가 이해할 수 있도록 표현한 개념화 명세서**

---

## 3. 공격 기법

* **사전 공격 (Dictionary Attack)**
  → ID/비밀번호로 사용될 가능성이 있는 단어 목록을 만들어 대입

* **무차별 공격 (Brute Force Attack)**
  → 가능한 모든 문자 조합을 무작위로 대입

* **ARP 스푸핑**
  → 공격자가 자신의 MAC 주소로 위조된 ARP Reply를 보내
  피해자의 ARP Cache를 변조하는 공격

---

## 4. 결합도 (Coupling) - "내공외제스자"

* **내용 결합도**
  → 다른 모듈 내부의 변수/기능을 직접 사용하는 경우

* **공통 결합도**
  → 전역변수를 공유하고 수정하는 경우

* **외부 결합도**
  → 외부 데이터 포맷, 통신 프로토콜, 장치 인터페이스 공유

* **제어 결합도**
  → 제어 신호를 통해 다른 모듈의 내부 동작을 제어

* **스탬프 결합도**
  → 배열, 객체 등 구조 전체를 전달

* **자료 결합도**
  → 단순 파라미터 전달로만 상호작용 (가장 바람직)
