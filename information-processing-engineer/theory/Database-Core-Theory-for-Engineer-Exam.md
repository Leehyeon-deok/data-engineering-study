1. 데이터베이스(DB)
데이터베이스(Database)

여러 사람이 사용할 데이터를 통합하여 저장하고 운영하는 시스템

특징
통합(Integrated) : 중복 제거
저장(Stored) : 컴퓨터 저장장치에 저장
운영(Operational) : 실제 업무에 사용
공용(Shared) : 여러 사용자가 공동 사용
암기

통저운공

2. DBMS 기능

DBMS(Database Management System)가 수행하는 기능

기능	설명
정의(DDL)	데이터 구조 생성·변경
조작(DML)	조회, 삽입, 수정, 삭제
제어(DCL)	권한, 보안, 무결성 관리
예시

학생 테이블 생성

CREATE TABLE Student(...)

→ 정의

학생 조회

SELECT *

→ 조작

권한 부여

GRANT SELECT

→ 제어

20. 정렬 알고리즘
삽입정렬(Insertion Sort)

이미 정렬된 부분에 삽입

예)

5 3 1

↓

3 5

↓

1 3 5

평균

O(n²)

선택정렬(Selection Sort)

가장 작은 값 선택

앞으로 이동

예)

5 2 8

↓

2 선택

↓

2 5 8

평균

O(n²)

버블정렬(Bubble Sort)

인접한 값 비교

큰 값 뒤로 이동

예)

5 3 2

↓

3 2 5

↓

2 3 5

평균

O(n²)

퀵정렬(Quick Sort)

기준(Pivot) 선택

작은 값 왼쪽

큰 값 오른쪽

재귀 반복

예)

7 3 9 2 5

Pivot=7

↓

3 2 5 | 7 | 9

평균

O(n log n)

최악

O(n²)
