🧠 1. 네트워크 보안
🔐 SSH (Secure Shell)
Telnet보다 강력한 보안 제공 원격 접속 프로토콜
기능:
원격 명령 실행
쉘 서비스 제공
특징:
데이터 암호화 전송
공개키 인증 방식 사용
기본 포트: 22번
💡 핵심
클라이언트 공개키를 서버에 등록
안전한 원격 접속 = SSH

🔑 3. 암호화 방식
🔐 대칭키 (비밀키)
암호화키 = 복호화키
✔ 특징
빠름 👍
키 관리 어려움 ❌
기밀성만 제공
✔ 키 개수

👉 n(n-1)/2

✔ 알고리즘
DES
AES
SEED
🔓 비대칭키 (공개키)
암호화키 ≠ 복호화키
✔ 특징
키 공유 필요 없음 👍
속도 느림 ❌
✔ 키 개수

👉 2n

✔ 알고리즘
RSA
디피-헬만 (Diffie-Hellman)
🔗 4. 객체지향 관계
✔ 포함 관계 (Aggregation / Composition)
객체가 다른 객체를 포함
📌 예시
자동차 → 엔진 포함
✔ 연관 관계 (Association)
객체 간 일반적인 연결 관계
📌 예시
학생 ↔ 강의 수강
✔ 의존 관계 (Dependency)
특정 객체가 다른 객체를 잠깐 사용
📌 예시
void print(Student s)

👉 Student를 일시적으로 사용

📊 6. LOC vs COCOMO
구분	LOC	COCOMO
기준	코드 줄 수	수학적 모델
목적	규모 측정	비용/기간 예측
시점	개발 후 정확	개발 전 예측
특징	단순	현실적
💡 한 줄

👉 LOC = 줄 수 / COCOMO = 비용 예측

🐍 7. Python 문법
✔ pass vs continue
for i in range(3):
    if i == 1:
        pass
    print(i)

👉 출력: 0 1 2

for i in range(3):
    if i == 1:
        continue
    print(i)

👉 출력: 0 2

💡 핵심
pass → 아무것도 안 함
continue → 아래 코드 스킵
🌐 8. 네트워크 핵심 개념
📡 IGMP (Internet Group Management Protocol)
멀티캐스트 그룹 관리 프로토콜
✔ 역할
어떤 호스트가 멀티캐스트 그룹에 참여할지 관리
📌 예시

👉 IPTV 방송
→ 여러 사용자에게 동시에 데이터 전송

🌍 NAT (Network Address Translation)
사설 IP → 공인 IP 변환 기술
✔ 목적
IP 주소 절약
보안 강화
📌 예시

👉 집에서 여러 기기 (폰, 노트북)
→ 하나의 공인 IP 사용

🧭 라우팅 프로토콜
네트워크에서 최적 경로 찾는 프로토콜
✔ 종류
RIP (거리 기반)
OSPF (링크 상태)
BGP (대규모 인터넷)
