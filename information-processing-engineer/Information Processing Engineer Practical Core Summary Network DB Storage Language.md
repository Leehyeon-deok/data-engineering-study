# 📘 Information Processing Engineer Practical Core Summary

## 🌐 VPN (Virtual Private Network)

👉 공중망(인터넷)을 이용하여 사설 네트워크를 구축하는 기술  
👉 물리적으로 떨어진 네트워크를 가상으로 연결

✅ 한줄 정리  
👉 인터넷 위에 만드는 사설 네트워크

---

## 🗄️ 관계대수 vs 관계해석

### 관계대수 (Relational Algebra)
👉 원하는 결과를 얻기 위한 **과정**을 정의 (절차적 언어)

### 관계해석 (Relational Calculus)
👉 원하는 결과만 정의 (비절차적 언어)

✅ 한줄 정리  
👉 관계대수 = 어떻게  
👉 관계해석 = 무엇을

---

## 📡 오류 제어 기법

### 패리티 검사 (Parity Check)
👉 1비트 추가해서 오류 검출

### CRC (순환 잉여 검사)
👉 다항식 기반 오류 검출  
👉 집단 오류 검출 가능

### 블록합 검사 (Block Sum)
👉 2차원 패리티 방식

### ARQ (Automatic Repeat reQuest)
👉 오류 발생 시 재전송 요청

✅ 한줄 정리  
👉 패리티(1비트) / CRC(강력) / 블록합(2차원) / ARQ(재전송)

## 💾 스토리지 시스템

구분   | DAS          | NAS             | SAN
------|--------------|------------------|-------------------
연결   | 직접 연결     | 일반 네트워크     | 전용 네트워크
공유   | 불가/어려움   | 파일 공유         | 블록 공유
속도   | 빠름          | 보통             | 매우 빠름
확장성 | 낮음          | 보통             | 높음
비용   | 저렴          | 중간             | 비쌈

✅ 한줄 정리  
👉 DAS → 직접 연결  
👉 NAS → 파일 공유  
👉 SAN → 고속 전용 네트워크

## 🐍 Python 함수

- map → 전체 변환
- filter → 조건 선택
- reduce → 누적 계산

✅ 한줄 정리  
👉 map(변환) / filter(선택) / reduce(누적)

## 🔒 final vs const

구분     | final (Java)        | const (C/C++)
--------|----------------------|-----------------------
의미     | 재할당 금지          | 값 변경 금지
포인터   | 없음                 | 있음
참조형   | 내부 값 변경 가능     | 상황에 따라 다름

✅ 한줄 정리  
👉 final = 주소 고정  
👉 const = 값 고정

## ☕ Java 문자열 + 연산

💣 핵심 공식  
👉 왼쪽부터 계산하면서  
👉 String 나오면 이후 전부 문자열 처리

### 예시
System.out.println("10" + 20 + 30); → 102030  
System.out.println(10 + 20 + "30"); → 3030  

✅ 한줄 정리  
👉 "String을 만나는 순간 끝났다"

💡 팁  
👉 ( )로 먼저 계산

System.out.println((10 + 20) + "30"); → 3030
