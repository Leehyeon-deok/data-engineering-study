🧠 3. 화이트박스 테스트 ⭐ (추가 핵심)
✅ 정의
프로그램 내부 구조(로직, 코드)를 기반으로 수행하는 테스트

👉 코드 흐름, 조건, 반복문 등을 직접 확인

✅ 주요 커버리지
커버리지	의미
문장 커버리지	모든 코드 한 번씩 실행
결정 커버리지	조건 결과 (T/F) 모두 실행
조건 커버리지	각 조건식의 T/F 모두 실행
조건/결정 커버리지	조건 + 결정 모두 만족
경로 커버리지	모든 실행 경로 수행
🔥 예시
if (A && B)
테스트	A	B	결과
1	T	T	T
2	F	T	F
3	T	F	F

👉 조건/결정 커버리지 만족

🚨 핵심 포인트
문장 → 코드 실행
결정 → 전체 조건 결과
조건 → 내부 조건 각각
경로 → 모든 흐름
🚀 한 줄 정리

👉
"화이트박스 = 코드 기준 / 커버리지로 테스트"

🌐 4. 오류 제어 방식
구분	내용

ARQ	오류 시 재전송
Parity	1비트 검사

🔥 암기
