## ✅ 1. 서브넷팅 (FLSM) 핵심 정리

### ✔ FLSM (Fixed Length Subnet Mask)

* 모든 서브넷의 크기와 마스크가 동일
* 네트워크를 동일한 크기로 나눔

### ✔ 서브넷 개수 공식

```
2ⁿ ≥ 필요한 서브넷 개수
```

* n = 빌린 비트 수

### ✔ 절차

1. 필요한 서브넷 개수 확인
2. 2ⁿ ≥ 개수 만족하는 최소 n
3. 새로운 서브넷 마스크 계산
4. 증가값 계산

### ✔ 증가값

```
256 - 마지막 옥텟 서브넷 마스크
```

### ✔ 핵심

* Subnet-Zero 사용 시 0번 서브넷 포함
* 미사용 시 첫 번째 서브넷 제외

## ✅ 2. UWB (Ultra Wideband)

### ✔ 개념

초광대역 무선 통신 기술

### ✔ 특징

* 매우 넓은 주파수 대역 사용
* 저전력
* 높은 정밀도 위치 측정 가능

### ✔ 활용

* 실내 위치 추적
* IoT
* 스마트키
* 자율주행

## ✅ 3. Kill Switch

### ✔ 개념

시스템 또는 소프트웨어를 긴급 상황에서 즉시 중단시키는 기능

### ✔ 목적

* 보안 사고 대응
* 시스템 오작동 방지
* 데이터 보호

### ✔ 예

* 악성코드 확산 차단
* 서비스 장애 시 기능 중단

## ✅ 4. 검증(Verification) vs 확인(Validation)

### ✔ 검증 (Verification)

* "우리가 제품을 제대로 만들고 있는가?"
* 개발 과정 중심
* 테스트 개발자, 시험자 시각

✔ 예

* 설계 검토
* 코드 리뷰
* 단위 테스트

---

### ✔ 확인 (Validation)

* "사용자가 원하는 제품인가?"
* 사용자 중심

✔ 예

* 사용자 테스트
* 시스템 테스트

---

## ✅ 5. strcat (C 언어)

### ✔ 개념

문자열을 뒤에 이어 붙이는 함수

### ✔ 사용

```c
#include <string.h>

char a[20] = "Hello ";
char b[] = "World";

strcat(a, b);
```

### ✔ 결과

```
Hello World
```

### ✔ 주의

* 대상 문자열 배열 크기 충분해야 함
* 버퍼 오버플로우 위험

## ✅ 6. 추상 클래스 (Abstract)

### ✔ 개념

구현되지 않은 추상 메서드를 포함한 클래스

### ✔ 특징

* 객체 생성 불가
* 상속받은 클래스에서 구현

### ✔ 목적

* 공통 구조 정의
* 다형성

### ✔ 예

```java
abstract class Animal {
    abstract void sound();
}

class Dog extends Animal {
    void sound() {
        System.out.println("멍멍");
    }
}
```

### ✔ 개념

애자일 개발에서 남은 작업량을 시각적으로 표현한 그래프

### ✔ 특징

* 시간 대비 남은 업무
* 프로젝트 진행 상태 파악

### ✔ 목적

* 일정 관리
* 문제 조기 발견

## ✅ 9. 제4정규형 (4NF)

### ✔ 개념

다치 종속(Multivalued Dependency) 제거

### ✔ 다치 종속

* 하나의 속성이 여러 독립 값을 가지는 경우

### ✔ 문제

데이터 중복 발생

### ✔ 해결

독립적인 속성으로 분리

---

### ✔ 예

학생

* 학생 → 취미
* 학생 → 자격증

취미와 자격증이 서로 독립

👉 테이블 분리

학생-취미
학생-자격증

✔ FLSM = 동일 크기
✔ UWB = 위치 측정
✔ Kill Switch = 긴급 중단
✔ strcat = 문자열 연결
✔ Abstract = 상속 후 구현
✔ 번다운 차트 = 남은 작업
