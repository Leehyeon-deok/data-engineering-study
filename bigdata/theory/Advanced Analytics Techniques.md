📌 고급 분석기법 정리 (Big Data Analyst Exam)
1️⃣ 분할표 (Contingency Table)

여러 범주형 변수를 기준으로 관측치를 기록한 표

범주형 변수 간 관계 분석에 사용

주로 카이제곱 검정, 오즈비(Odds Ratio) 계산에 활용

👉 예시

성별 vs 질병 여부

흡연 vs 폐암 발생

2️⃣ 다변량 분석 (Multivariate Analysis)

여러 개의 변수들을 동시에 분석

변수 간 관계를 파악하고 차원을 축소

📌 목적

변수 구조 파악

데이터 축소

패턴 발견

3️⃣ 요인분석 (Factor Analysis)

다수 변수 간 상관관계를 분석하여 소수 요인으로 축약

기술통계 기반

독립/종속 변수 개념 없음

같은 특성을 가진 변수끼리 묶음

요인 간 서로 독립

📌 요인 추출 방법

주성분 분석 (PCA)

공통요인 분석

📌 요인 회전

해석력을 높이기 위해 수행

4️⃣ 시계열 분석 (Time Series Analysis)

시간 흐름에 따라 관찰된 데이터를 분석하고 미래 예측

📌 시계열 특징
✔ 정상성 (Stationarity)

평균과 분산이 일정해야 함

✔ 차분 (Differencing)

현재값 – 이전값

정상성 확보 목적

✔ 자기상관 (Autocorrelation)

과거와 현재가 연관

✔ 백색잡음 (White Noise)

오차항

독립적인 랜덤 신호

📌 시계열 모형
✔ 자기회귀 AR

과거값으로 미래 예측

✔ 이동평균 MA

백색잡음의 선형결합

✔ 지수평활법

최근 데이터에 높은 가중치

✔ ARIMA

AR + I + MA

(p, d, q)

d = 차분 횟수
