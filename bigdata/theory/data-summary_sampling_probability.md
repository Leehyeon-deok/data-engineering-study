📊 데이터 요약 · 표본추출 · 확률분포 핵심 정리

(빅데이터분석기사 대비)

1️⃣ 데이터 요약 (Data Summarization)
📌 기술통계(Descriptive Statistics)

데이터를 요약·묘사·설명하는 통계 기법

데이터 특성을 정량화하여 파악

📌 기술통계의 분류

중심화 경향

평균(Mean)

중앙값(Median)

최빈값(Mode)

분산도

분산(Variance)

표준편차(Standard Deviation)

범위(Range)

사분위수(IQR)

분포의 형태

왜도(Skewness)

첨도(Kurtosis)

2️⃣ 모집단과 표본

모집단(Population)
: 관심 대상의 전체 집합

표본(Sample)
: 모집단의 특성을 추정하기 위한 일부 자료

표본추출(Sampling)
: 모집단으로부터 표본을 선택하는 과정

3️⃣ 조사 방법
📌 전수조사

모집단 전체를 조사

정확하지만 시간·비용 큼

📌 표본조사

일부 표본만 조사

결과를 모집단 전체로 추정

시간·비용 효율적

4️⃣ 표본추출 오차

표본이 모집단 특성을 과잉 대표 / 과소 대표할 때 발생

표본 수가 적을수록 커짐

5️⃣ 확률 표본추출 기법

모든 추출 단위에 사전 확률이 주어짐

🔹 단순 무작위 추출

난수 사용

가장 기본적인 방법

편향 최소화

🔹 계통 추출

일정 간격(k)으로 추출

시작점은 무작위

🔹 층화 추출

모집단을 겹치지 않는 층으로 나눔

각 층에서 무작위 추출

이질적인 모집단에 적합

🔹 군집 추출

모집단을 군집으로 나눔

일부 군집을 선택 후 전체 조사

비용 절감 목적

6️⃣ 비확률 표본추출 기법

추출 확률을 객관적으로 알 수 없음
모집단 정의가 어려운 경우 사용

🔹 간편 추출

조사자의 편의에 따라 선택

예: 길거리 설문조사

🔹 판단 추출

조사자의 경험과 판단에 의존

🔹 할당 추출

미리 정한 비율만큼 표본 추출

🔹 눈덩이 추출

접근이 어려운 집단 조사

기존 응답자가 다음 응답자 소개

7️⃣ 이산 확률분포
🔹 베르누이 분포

결과: 성공 / 실패

성공 확률 p

🔹 이항 분포

베르누이 시행을 n번 독립 반복

평균: np

분산: np(1-p)

🔹 다항 분포

가능한 결과가 2개 이상

🔹 포아송 분포

단위 시간(공간) 내 사건 발생 횟수

평균 = 분산 = λ

🔹 기하 분포

처음 성공할 때까지의 시행 횟수

🔹 음이항 분포

r번째 성공까지의 시행 횟수

🔹 초기하 분포

비복원 추출

모집단 크기가 작을 때 사용

8️⃣ 연속 확률분포
🔹 연속균등분포

모든 구간에서 확률 동일

🔹 지수분포

포아송 과정에서

다음 사건까지의 대기시간

기억 없음 성질(Memoryless)

🔹 정규분포

평균 = 중앙값 = 최빈값

좌우 대칭 종 모양

🔹 표준정규분포

평균 0, 분산 1

🔹 감마분포

지수분포의 일반화

🔹 카이제곱 분포

k개의 표준정규변수 제곱합

분산 분석, 적합도 검정

🔹 스튜던트 t 분포

표본 수가 적을 때 평균 추정

자유도 = n - 1

🔹 F 분포

두 분산의 비율

분산 분석(ANOVA)

9️⃣ 표본분포 & 중심극한정리
📌 표본분포

표본 통계량(평균 등)의 확률분포

📌 중심극한정리 (CLT)

동일 분포를 가진 독립 확률변수 n개의 평균은
👉 n이 충분히 크면 정규분포에 근사

📌 시험 핵심 문장

모집단 분포와 무관하게 표본 평균의 분포는 정규분포에 수렴

🔟 시험 대비 공식 요약

평균: μ

분산: σ²

표준편차: σ

이항분포 평균: np

이항분포 분산: np(1-p)

포아송 평균 = 분산 = λ

t분포 자유도: n-1

기술통계 → 표본추출 → 확률분포 → 중심극한정리 흐름을 이해하면 합격
