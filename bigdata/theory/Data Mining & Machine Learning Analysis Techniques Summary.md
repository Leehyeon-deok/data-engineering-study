📊 데이터 분할 및 분석기법 정리 
✅ 1. 데이터 분할 (Data Split)
✔ 목적

과적합(Overfitting) 방지

과소적합(Underfitting) 방지

모델 일반화 성능 향상

데이터 불균형 문제 대응

✔ 일반적인 비율

훈련용 (Train) : 50%

검증용 (Validation) : 30%

평가용 (Test) : 20%

📌 실무에서는 70:15:15 또는 80:10:10도 많이 사용됨.

✔ 과대적합 vs 과소적합

과대적합 : 훈련 데이터에는 잘 맞지만 새로운 데이터 예측 실패

과소적합 : 데이터 패턴을 충분히 학습하지 못함

👉 해결 방법

데이터 증가

규제 적용

모델 단순화

교차검증(K-Fold)

✅ 2. 회귀 분석 (Regression)
✔ 개념

독립변수가 종속변수에 미치는 영향을 분석하고 미래 값을 예측하는 기법
예측선(회귀선)을 통해 실제값과 차이를 최소화함.

✔ 종류

단순 회귀

다중 회귀

✔ 선형 회귀 가정 (정처기 중요 ⭐)

선형성

독립성

정규성

등분산성

비상관성 (다중공선성 없음)

✅ 3. 변수 선택 방법
✔ 전진 선택법

변수를 하나씩 추가

성능이 좋아지면 유지

✔ 후진 제거법

모든 변수 포함 후 제거

✔ 단계적 선택법

전진 + 후진 결합

✔ 벌점 기반 선택

AIC, BIC 등 사용

변수 추가 시 복잡도 고려

✅ 4. 로지스틱 회귀 분석

종속변수가 범주형일 때 사용

결과는 확률값 (0~1)

예 : 합격/불합격, 질병 여부

✅ 5. 의사결정나무 (Decision Tree)

데이터를 여러 기준으로 분할

최종 분류 결과 도출

해석이 쉬움

과적합 위험 존재
