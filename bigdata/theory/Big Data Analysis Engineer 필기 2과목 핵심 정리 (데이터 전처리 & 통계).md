📌 1. 데이터 전처리 (Data Preprocessing)
✅ 데이터 정제
결측값 처리 / 이상치 제거 → 데이터 신뢰도 향상
주의: 차원축소는 정제 과정이 아님
✔ 결측치 처리 방법
평균 대치 → 변동성 감소 (⚠ 표준오차 과소추정)
최빈값 → 명목형 데이터 (예: 혈액형)
최근접 대치 (KNN)
단순확률대치 → 랜덤 값 선택

👉 ❗ 시험 포인트

“삭제만 한다” → ❌
“데이터 특성 고려” → ⭕
✅ 이상치 vs 노이즈
이상치: 실제 값이지만 극단값
노이즈: 오류로 생성된 값 (측정/처리 오류)

👉 시험 핵심

측정 오류 → 이상치로 처리 가능 ⭕
✅ 스케일링 (Scaling)
Min-Max 정규화
Z-score 표준화 → (X - 평균) / 표준편차
Robust Scaling → 이상치 영향 적음
✅ 인코딩
원-핫 인코딩 → 차원 증가 (차원의 저주 유발)
타깃 인코딩 → 평균 활용 (표준편차 X)
✅ 파생변수
기존 변수로 새로운 변수 생성

👉 ❗ 주의

종속변수 사용 → 데이터 누설 발생
✅ 변수 변환
로그 / 제곱근 → 양의 왜도 해결
제곱 → 음의 왜도 해결
✔ Box-Cox 변환
목적: 정규분포 근사
조건: 양수 데이터만 가능

✅ 차원의 저주

차원축소 활용
📌 3. 기초 통계
✅ 평균 종류
산술 평균 → 이상치 영향 큼
기하 평균 → 성장률 (금융, 물가)
조화 평균 → 속도
✅ 분산
표본분산: n-1
모분산: n

👉 이유: 불편추정량

✅ 표준오차
표준편차 / √n
👉 표본 수 증가 → 오차 감소
✅ 중심극한정리 (CLT)
표본 크기 충분히 크면 → 정규분포
모집단 분포와 무관
📌 4. 분포
✅ 정규분포
평균 = 중앙값 = 최빈값
✅ 포아송 분포
평균 = 분산
단위 시간 사건 수

예:

콜센터 전화 수
페이지 오탈자 수
✅ 이항 → 포아송 근사
n 크고 p 작을 때
✅ 초기하 분포
비복원 추출
독립 X
📌 5. 상관 & 공분산
✅ 공분산
0 → 독립 아닐 수도 있음
독립 → 반드시 0
✅ 상관계수
-1 ~ 1
제곱 → 결정계수
✅ 종류
피어슨 → 연속형
스피어만 → 순위 (이상치 영향 적음)
Phi / 크래머 V → 명목형
📌 6. 검정
✅ Z vs T
Z: 모표준편차 O
T: 모표준편차 X
✅ 유의수준
0.05 → 5% 오류 허용
신뢰수준	Z값
90%	1.645
95%	1.96
99%	2.58
✅ p-value
p < α → 귀무가설 기각
✅ 검정 종류
대응표본 → 전후 비교
ANOVA → 3개 이상 평균 비교
F검정 → 분산 비교
카이제곱 → 범주형
📌 7. 왜도 & 분포 형태
왜도 > 0 → 오른쪽 꼬리
👉 평균 > 중앙값 > 최빈값
📌 8. 연관분석
지지도 = A∪B / 전체
신뢰도 = A∪B / A

📌 9. 모델 평가

📌 10. 핵심 암기 포인트 (시험 직전)

✔ 표본분산 → n-1
✔ 포아송 → 평균 = 분산
✔ PCA → 분산 최대 방향
✔ Box-Cox → 양수만
✔ 원핫 → 차원 증가
✔ 공분산 0 ≠ 독립
✔ p-value < α → 기각
✔ Z vs T → 표준편차 유무
