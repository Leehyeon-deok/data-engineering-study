1. 분석모형 선정의 필요성

분석 기법이나 분석 알고리즘을 적용하기 전에 적절한 분석모형을 선정하는 과정이 반드시 필요하다.
이를 위해 먼저 분석이 필요한 데이터의 속성을 세부적으로 파악하고 전처리한 후, 분석 목적에 부합하는 분석모형을 선택해야 한다.

분석 목적 설정

분석모형 선정의 출발점은 분석 목적 정의이다.

의사결정 시 불확실성 해소

데이터 요약 및 패턴 파악

원인과 결과의 인과관계 분석

미래 결과에 대한 예측

3. 분석모형의 정의와 종류
1️⃣ 예측 분석모형 (Predictive Model)

목적: 앞으로 어떤 일이 발생할 것인가?

과거 데이터를 기반으로 미래 결과 예측

예: 수요 예측, 이탈 예측, 매출 예측

2️⃣ 현황 진단 분석모형 (Descriptive / Diagnostic Model)

목적:

과거에 어떤 일이 있었는가?

왜, 어떻게 발생했는가?

현재 상태는 어떠한가?

예: 매출 감소 원인 분석, 사용자 행동 분석

3️⃣ 최적화 분석모형 (Prescriptive Model)

목적: 어떻게 하면 원하는 결과를 얻을 수 있는가?

의사결정 대안을 제시

예: 자원 배분 최적화, 일정 최적화

4. 분석모형 정의를 위한 사전 고려사항

분석을 진행하기 전, 분석모형 정의의 타당성을 판단해야 한다.

사전 판별 기준

평가 기준표(Score Table)를 작성

항목별 점수 부여 후 총점으로 분석 가능성 판단

주요 평가 기준 예:

추진 시급성

구현 가능성

데이터 확보 가능성

※ 실무에서는 추진 시급성 + 구현 가능성만으로도 사전 판별 기준으로 활용 가능

5. 데이터 및 접근 방법 검토
데이터 확보 여부 판단

분석모형 정의에 필요한 데이터가 충분히 존재하는지 확인

관련 과거 분석 사례 또는 기존 솔루션 활용 가능성 검토

분석 접근 방식
1️⃣ 상향식 접근 (Bottom-Up)

문제 정의가 어려운 경우

대량의 데이터를 분석하여 패턴 및 인사이트 도출

2️⃣ 하향식 접근 (Top-Down)

문제 정의가 가능한 경우

문제 탐색 후 여러 분석모델을 적용하여 해결책 도출

9. 분석모델링 설계와 검정

분석 목적에 따라 가설 검정 기반 모델링을 수행한다.

1️⃣ 유의수준 결정 및 가설 설정

귀무가설(H₀)

표본 관찰을 통해 모집단은 특정 특성을 가진다고 가정

가설 검정은 귀무가설이 참이라는 가정에서 시작

대립가설(H₁)

귀무가설이 기각될 경우 채택되는 가설

2️⃣ 검정통계량 설정

가설을 검정하기 위한 기준 값

확률분포 상 위치에 따라 귀무가설 채택 여부 결정

3️⃣ 기각역 설정

확률분포 상에서 귀무가설을 기각하는 영역

4️⃣ 검정통계량 계산

표본 데이터를 이용하여 실제 검정통계량 산출

5️⃣ 통계적 의사결정

검정통계량이 기각역에 포함되면 귀무가설 기각

포함되지 않으면 귀무가설 채택(기각하지 않음)
