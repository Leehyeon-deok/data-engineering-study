3. 데이터 분할(Data Splitting)

모델 구축 및 성능 평가를 위해 전체 데이터를 분리

데이터 구성

학습 데이터(Training Data)
→ 모델 학습

평가 데이터(Validation Data)
→ 모델 튜닝 및 성능 비교

검증 데이터(Test Data)
→ 최종 성능 검증

📌 목적:

모델의 일반화 성능 평가

4. 과대적합(Overfitting)
개념

학습 데이터에 과도하게 최적화된 모델

학습 데이터에서는 정확도 높음

새로운 데이터에서는 예측 성능 저하

발생 원인

모델이 너무 복잡함

학습 데이터에 노이즈까지 학습

방지 방법

데이터 분할

K-Fold 교차검증

정규화(Regularization)

모델 단순화

5. 과소적합(Underfitting)
개념

모델이 너무 단순하여

데이터의 패턴이나 규칙을 제대로 학습하지 못함

특징

학습 데이터에서도 성능 낮음

예측 정확도 전반적으로 낮음

6. 일반화(Generalization)
개념

학습 데이터를 통해 생성된 모델이

평가 데이터뿐만 아니라

검증용 테스트 데이터에서도 안정적인 성능을 보이는 상태
