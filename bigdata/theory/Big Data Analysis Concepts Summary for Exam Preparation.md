3. 머신러닝 개념
✔ 지도학습 / 비지도학습 / 강화학습
구분	설명	예시
지도학습	정답 있음	분류, 회귀
비지도학습	정답 없음	군집
강화학습	보상 기반 학습	게임 AI
✔ 회귀 (Regression)
독립변수가 종속변수에 미치는 영향 분석

📌 예시

공부시간 → 시험점수
✔ 분류 vs 회귀
분류: 결과가 범주형
→ 랜덤포레스트, SVM, 로지스틱 회귀
회귀: 결과가 연속형
→ 다중회귀분석
✔ GAN (Generative Adversarial Network)
생성모델 + 판별모델 구조

📌 예시
