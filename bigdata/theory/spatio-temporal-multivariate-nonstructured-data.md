1️⃣ 시공간 데이터 (Spatio-Temporal Data)
🔹 시공간 데이터 개념

공간적 정보(위치, 형태 등) 에 시간의 흐름(이력 정보) 이 결합된
다차원 데이터

시간 + 공간의 변화 양상을 동시에 분석 가능

🔹 시간 데이터 (Temporal Data)

특정 한 시점에 대한 스냅샷 정보

시간에 따른 변화, 추세, 패턴 분석에 활용

예: 매출 시계열, 주가 데이터

🔹 공간 데이터 (Spatial Data)

위치, 거리, 면적, 형태 등의 정보를 포함

기존 DB보다 구조가 복잡하고 다양한 값을 가짐

초점: 효율적인 관리 · 저장 · 검색 · 활용

공간 데이터 모델

관계형 모델

객체지향 모델

🔹 시공간 데이터

시간 데이터 + 공간 데이터의 결합 형태

시간에 따른 위치 변화 분석 가능

🔹 시공간 데이터 활용 기술

지리정보 시스템(GIS)

위치 기반 서비스(LBS)

차량 위치 추적 서비스

물류 추적, 스마트 시티, 재난 관리

2️⃣ 다변량 데이터 탐색 (Multivariate Data Analysis)
🔹 개념

여러 변수 간의 인과관계 및 상관관계를 분석

목적:

변수 축약

개체 분류

패턴 분석

예측 모델 구축

🔹 종속변수와 독립변수 간 인과관계 분석
✅ 다중 회귀 분석 (Multiple Regression)

독립변수가 2개 이상인 회귀 모형

각 독립변수는 종속변수와 선형 관계를 가진다고 가정

✅ 로지스틱 회귀 (Logistic Regression)

독립변수의 선형 결합을 이용

사건 발생 확률 예측 (0~1)

이진 분류 문제에 사용

✅ 분산분석 (ANOVA)

3개 이상 집단 평균 차이 검정

표본 평균 간 분산 vs 표본 내 분산 비교

가설 검정 기법

✅ 다변량 분산분석 (MANOVA)

종속변수가 2개 이상

여러 반응 변수의 평균 차이 동시 검정

3️⃣ 공분산과 독립성 관계
🔹 공분산

두 변수가 함께 변하는 정도

🔹 관계 정리

독립적인 변수 → 공분산 = 0

공분산 행렬의 비대각 원소 = 0

❗ 공분산이 0이라고 해서 항상 독립은 아님

🔹 두 확률분포 간 독립성 확인 방법

분포 형태 확인

공분산 및 상관계수 분석

독립성 검정 수행

4️⃣ 변수 축약 (Dimension Reduction)
🔹 개념

변수 간 상관관계를 이용해 변수 수 감소

정보 손실 최소화가 목표

🔹 주요 기법

주성분 분석(PCA)

요인 분석(Factor Analysis)

정준 상관 분석(Canonical Correlation Analysis)

5️⃣ 개체 유도 및 분류
🔹 개체 유도

개체 특성을 측정한 변수 간 강한 상관관계 활용

유사한 개체들을 그룹화

🔹 군집 분석 (Clustering)

사전 정보(라벨) 없이 수행

관측값 간 거리 기반 분류

비지도 학습

🔹 다차원 척도법 (MDS)

고차원 데이터를 저차원 공간에 시각화

거리 관계 유지가 목적

🔹 판별 분석

집단 정보가 있는 경우

새로운 관측치를 어느 집단에 속하는지 분류

6️⃣ 비정형 데이터 탐색 분석
🔹 데이터 마이닝 (Data Mining)

대규모 데이터에서 패턴·규칙·지식 추출

❗ 단점

데이터에 과도하게 의존

현실을 충분히 반영하지 못하면
→ 잘못된 모형 구축 위험

🔹 텍스트 마이닝 (Text Mining)

자연어 처리(NLP) 기반

문서, 문장 내 숨겨진 의미 및 주제 추출

🔹 오피니언 마이닝 (Opinion Mining)

텍스트의 감정, 태도, 뉘앙스 분석

텍스트 마이닝보다 감성 중심

🔹 웹 마이닝 (Web Mining)

웹 자원으로부터 의미 있는 패턴과 추세 도출

로그 데이터, 웹 문서, 링크 구조 분석

✅ 전체 요약

시공간 데이터 = 공간 + 시간

다변량 분석 = 변수 간 관계 분석

변수 축약 = 정보 유지하며 차원 감소

군집 분석 = 라벨 없는 분류

비정형 데이터 = 텍스트·웹 중심 분석
