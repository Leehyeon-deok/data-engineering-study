📘 BigData Analysis - Subject 3: Modeling (빅데이터 모델링 정리)
1. 머신러닝 기본 개념
✔ 지도학습 vs 비지도학습
지도학습
입력(X) + 정답(Y) 존재
예: 회귀, 분류
비지도학습
정답 없음
예: 군집분석 (K-means)
2. 분류 / 회귀 모델
✔ 로지스틱 회귀 (Logistic Regression)
종속변수: 범주형
확률 기반 분류 모델
출력: 0 ~ 1

👉 핵심 개념

오즈(Odds) = p / (1 - p)
로짓(Logit) = log(odds)

👉 예시

이메일이 스팸일 확률 예측
✔ 선형 회귀
종속변수: 연속형
평가 지표:
MSE
결정계수 (R²)
✔ 의사결정나무 (Decision Tree)
분류 + 회귀 둘 다 가능

👉 분리 기준

분류: 지니지수, 엔트로피
회귀: 분산 감소, F통계량

👉 특징

해석이 쉬움
과적합 위험 있음
✔ SVM (서포트 벡터 머신)
초평면으로 데이터 분리

👉 핵심

커널 함수 → 비선형 가능
RBF(가우시안) 커널 사용

👉 하이퍼파라미터

C: 규제 강도
작으면 과소적합
gamma: 영향 범위
크면 과적합 가능
