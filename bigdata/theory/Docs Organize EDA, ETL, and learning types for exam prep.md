✔ 핵심 특징
인공신경망 기반 모델
Hidden Layer가 많을수록 복잡한 패턴 학습 가능
모델 해석이 어려움 (Black Box)
✔ 주요 개념
Activation Function
Sigmoid
ReLU (가장 많이 사용)
학습 방법
역전파 (Backpropagation)
정규화 기법
Dropout
→ 일부 뉴런을 랜덤하게 제거하여 과적합 방지

4️⃣ 머신러닝 학습 유형
유형	설명
지도학습	정답(label)이 있음 → 분류, 회귀

🔥 시험 핵심 요약 (한 줄 암기)
ETL = 추출 → 변환 → 적재
딥러닝 = 해석 어려움 + 역전파 + ReLU
분류/회귀 = 지도학습
보상 = 강화학습
t-closeness = 민감정보 분포 보호
EDA = 데이터 이해 (모델 X)
성숙도 = 도입 → 활용 → 확산 → 최적화
