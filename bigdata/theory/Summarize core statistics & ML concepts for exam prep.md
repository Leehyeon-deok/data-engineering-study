📊 Big Data Analysis – Key Concepts (Mock Exam Notes)
1. Stepwise Selection (단계적 선택법)
📌 개념

전진선택법 + 후진제거법을 결합한 변수 선택 방법

🔄 과정
전진선택법 (Forward Selection)
→ 가장 유의한 변수를 하나씩 추가
후진제거법 (Backward Elimination)
→ 추가 후, 유의하지 않은 변수 제거
반복
제거된 변수는 다시 포함하지 않음
더 이상 유의한 변수가 없을 때 종료
📌 핵심 포인트
모델 성능 + 변수 수 균형 맞추기
과적합 방지
💡 예시
초기: 변수 없음

Step 1: X1 추가 (p < 0.05)
Step 2: X2 추가 → X1 p값 상승 → X1 제거
Step 3: X3 추가 → 유지

최종: X2, X3
2. Oversampling (오버샘플링)
📌 개념

소수 클래스 데이터를 복제하여 데이터 불균형 해결

📌 목적
모델이 소수 클래스를 무시하는 문제 해결
재현율(Recall) 향상
💡 예시
클래스 A: 100개
클래스 B: 10개

→ B를 복제해서 100개로 맞춤
⚠️ 단점
과적합 위험 증가
3. Spearman Correlation (스피어만 상관계수)
📌 개념

값이 아니라 순위(rank) 기반 상관계수

📌 특징
비선형 관계도 파악 가능
이상치(outlier)에 강함
표본이 적을 때 유리
💡 예시
점수: [90, 80, 70]
→ 순위: [1, 2, 3]

두 변수의 순위 비교 → 상관계수 계산
4. Z값 (표준정규분포 임계값)
신뢰수준	Z값
90%	1.645
95%	⭐ 1.96
99%	2.576
📌 사용 조건
표본 크기 n ≥ 30
모분산을 알거나 정규성 가정 가능
5. Curse of Dimensionality (차원의 저주)
📌 개념

차원(변수 수)이 증가할수록 데이터가 희소해지는 현상

📌 문제점
거리 계산 의미 감소
과적합 발생
모델 성능 저하
💡 예시
변수 2개 → 데이터 밀집
변수 100개 → 데이터 퍼짐 (희소)
6. Stratified Sampling (층화 추출)
📌 목적
모집단 특성을 잘 반영
분산 감소 → 더 정확한 추정
📌 표본 배정법
1) 비례 배분법
각 층의 비율 그대로 샘플링
A층 70%, B층 30%
→ 샘플도 70:30
2) 네이만 배분법 (Neyman Allocation)
표준편차 고려
분산이 큰 층에 더 많이 배정
3) 최적 배분법
비용 + 분산 함께 고려
7. Probability Distributions (확률분포)
📌 이산확률분포 (Discrete)

👉 베포항하이

분포	설명
베르누이	성공/실패
포아송	일정 시간 내 사건 발생 횟수
이항	여러 번 시행에서 성공 횟수
하이퍼지오메트릭	비복원 추출
📌 연속확률분포 (Continuous)

👉 지수, 카이, F, t, 정규

분포	설명
정규분포	평균 중심 대칭
지수분포	사건 발생 간 시간
카이제곱	분산 분석
t분포	소표본 평균 검정
F분포	분산 비교
